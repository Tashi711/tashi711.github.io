记疫

发表于 2020-04-18 | 分类于 奇幻の私 ， ノーマルジブン | | 阅读次数:

ねぇ、キオク

相信2020年的春天让所有中国人甚至全世界人都难以忘记，用一点字形来记录下对这次疫情的记忆。

1月18号，前几天刚帮导师批完数逻大程，看到一个宝藏up主，感慨了下：「其实大家都在为自己不为大多数人知的目标而努力着，世界很大，自己在自己的世界忙碌，与大部分人没有联系，只有着一张属于自己的小小的网络，而这些网络，编织成了这个世界，偶然间，通过一条神秘的线，看到了远离自己网络的某一端，才发现世界是一样精彩着，也平凡着，……，平凡并不是一件简单事，希望自己这一年，慢慢的理解与学会平凡吧」。然后是看到赵忠祥去世的的消息，感叹各大平台控评的能力。然后18号这天，也是自己选择回家的日子，其实离春节已经没几天了，但是心里其实想着少在家待几天，就多在学校留了几天，这天应该是还没有什么大的动静，仅仅记得是在知乎上看到一些说法，然后在某些群里面也有一点点消息在传，只是没有人把这个当一回事，回家这天，自己这趟动车，自己应该是只能看见自己一个人带了口罩，当然并不是听闻这些消息，仅仅是因为自己最近养成的长途人口密集地戴口罩的习惯而已。

七年之旅，我于浙大的二三事

发表于 2020-04-02 | 分类于 奇幻の私 ， ノーマルジブン | | 阅读次数:

リンクボーナス（バンガイ）

2020/02/02：为了产生リンク进行比较猥琐的分类。写着写着，时间从2月1号跳到了2月2号，新年伊始已经过去了一个月，这个日子也蛮有感觉的，只是这场灾难，毁灭了包含它在内的太多美好的东西。想想，本科以来确实没文字记录，不觉行将毕业，这篇文章讲讲这近7年的记忆片段，争取在毕业那天发表出来。

2020/02/16：基本每天写一段，数了下，9段多，今天写了两段，也就是写了不知不觉8天了，当然每天也就写个一两个小时，感觉有点享受这种晚上安静写点东西的感觉了。准备停一下，到了最后快毕业了完成最后一段，这里面每一段都算是现实的一段人生阶段吧，当然感觉不同天写的，最后读起来一定很精分吧，可能需要好好改一下。

2020/04/02：正式完稿撒花。

ねぇ、ZJU

记得那是2013年的暑假，时间过得真快，得到自己被浙江大学录取的信息的那一刻，就好像发生在昨天一样。其实已经忘记了那个时候的心境，但是可以肯定的是，我非常的平静，家里人发了疯似的到处打电话报喜，还要我也打，我很不情愿的打了几个电话，因为真心觉得没有必要，反而感觉是像在向全世界炫耀。至于为什么自己这么平静，大概是只有自己知道这个结果其实是意料之中的match，既然是match而不是beyond或者below，那就没有过于开心或者过于难过的必要了。浙大是第二志愿，第一志愿是复旦，只是算上竞赛或者少数民族的加分自己还差了两分，而浙大后面填的那几个学校，大概也不太会考虑，如果进不了浙大，大概会选择直接复读吧。知道自己录取结果不久，高一的班主任x老师就打电话过来了，他那个时候被分到了下一届高三做年级主任了，打电话回来给我是因为觉得我复读一定能上清北，希望我回去冲一冲，我与x老师的故事太复杂了，这里就不细说了，记得当时我非常坚定的拒绝了，直到现在我也非常欣赏那个时候我的决断。那个夏天还发生了好多事吧，但就那样后知后觉的，开启了与浙大的长达七年的缘分。
