「真假硬币」——一道有趣的概率题

发表于 2019-08-19 | 分类于 编程 ， 其他 | | 阅读次数:

题目

在看面筋时遇到这样一道题：真硬币m个，假币n个。假币只有正面。真币投掷正面概率为p。其中某硬币投掷k次都是正面，求它为真币概率。

分析

想了蛮久，才发现这是道条件概率。P(x=1 | k) = P(x=1 and k) / P(k)。其中P(x=1 and k) = m/(n+m) * p^k，P(k) = n/(n+m) + m/(n+m) * p^k。
