「线段树」进化

发表于 2019-06-10 | 分类于 编程 ， 数据结构与算法

简述

洛谷模板题3373 - 线段树 2。
基本就是加强了前一个写的线段树，支持乘法，添加取模运算。
另外屈服于vector的低效率，再次验证了大数据静态开数组的必要性。
基本思路也比较简单，加法与乘法各打一个懒标记，基本框架跟只有加法一样。
只是细节：有加法时直接加，而乘法需要将加法的懒标记乘上去。
