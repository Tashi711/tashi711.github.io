「LeetCode 4. Median of Two Sorted Arrays」解题报告

发表于 2018-10-09 | 分类于 编程 ， 解题报告 ， LeetCode | | 阅读次数:

题目信息

  • 链接：LeetCode 4. Median of Two Sorted Arrays
  • 来源：LeetCode
  • 难度：Hard

题目分析

描述

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2 。
请找出这两个有序数组的中位数。要求算法的时间复杂度为 O(log (m+n)) 。
你可以假设 nums1 和 nums2 不同时为空。

示例 1:
nums1 = [1, 3]
nums2 = [2]
中位数是 2.0

示例 2:
nums1 = [1, 2]
nums2 = [3, 4]
中位数是 (2 + 3)/2 = 2.5

解题思路

基本按照官方题解走就行了。

复杂度分析

时间复杂度为$O(\log(m+n))$。
空间复杂度为$O(1)$。
