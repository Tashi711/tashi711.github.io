云顶之弈第三赛季小结

发表于 2020-08-29 | 分类于 オモシロサ ， ニチジョウ

前记

漫长的第三赛季结束了，以钻石守门的成绩作为这个赛季的成绩单吧。其实拳头从这个赛季开始已经设定大的赛季分成两个小段了，所以这个赛季前半段还有个黄金守门的成绩，不过当时没时间冲分，所以守个门拿个赛季奖励而已。之后争取每个小赛季都至少钻石守门吧，其实自己最近的比赛排到的都是大师和钻1分段的，所以其实隐藏分已经上去了，只是需要时间与精力去冲分了。

回顾

看了下数据，这个赛季的总场次玩的很少，毕竟工作了，但是也说明自己的对局质量蛮不错的，前半段19场吃鸡率26.3%，后半段35场吃鸡率14.3%，吃鸡的阵容，从前半段一开始的全员奥德赛（吃了4局），赌霞（吃了1局），源计划（吃了2局），到后半段狙神（吃了1局），女团（吃了2局），银河机甲（吃了2局），看得出来吃鸡越来越难了，大部分靠着2、3名吃分，因为个人觉得这个赛季非常的平衡了，没有哪个阵容完全统治，都有相应的克制阵容或者克制装备等等。排位之外自己也用前半段的奥德赛、赌霞、源计划和后半段的狙神、女团吃了很多局，偶尔也玩玩暗星、6法瑞文、6剑、战地之类的，不过玩海盗从来没有吃过。

评价下这个版本，总的来说感觉是一个非常优秀的版本，有很多创新，总体比前两个赛季都完善很多，更新地图，以及各种各样的小星系增添了很多游戏的可玩性。最关键的是个人觉得这个两个小赛季的设计非常的舒服，因为按之前的赛季看太短了，感觉好不容易熟悉适应一个版本之后马上就赛季末了，这个设定可以让像我这样平时玩的不多的玩家有充分的缓冲适应时间。

感想与展望

目标就是下个赛季冲个大师，平时少玩点匹配多玩点排位，毕竟感觉平时匹配胜率居然还比排位高很多。
